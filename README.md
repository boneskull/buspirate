# buspirate

> Might be a GUI for the [Bus Pirate](http://dangerousprototypes.com/docs/Bus_Pirate) one day


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
